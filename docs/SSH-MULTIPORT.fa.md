# راه‌اندازی SSH چندپورته (systemd + UFW)

**این راه‌اندازی از قبل در SSH Plus Manager پیاده‌سازی شده است.** اسکریپت، OpenSSH، فعال‌سازی سوکت systemd و UFW را برای شما پیکربندی می‌کند. برای افزودن/حذف پورت‌ها، اجرای راه‌اندازی سریع چندپورته یا بررسی پیکربندی، از **menu** → **[15] CONNECTIONS** → **SSH Multi-Port Setup** استفاده کنید—مگر این‌که خودتان بخواهید فایل‌ها را به‌صورت دستی ویرایش کنید.

روی سیستم‌های مدرن Debian/Ubuntu با استفاده از **ssh.socket** تست شده است.

موارد استفاده:

- اتصال SSH روی شبکه‌های محدود کننده (مثل ۴۴۳، ۸۰)
- پورت‌های جایگزین اضافی (مثلاً ۳۳۹۸، ۸۰۸۰، ۸۴۴۳)
- تونل‌سازی یا مدیریت مبتنی بر SSH روی پورت‌های غیراستاندارد

---
۱. **menu** را اجرا کنید.

۲. **[08] CONNECTIONS** را انتخاب کنید.

۳. **SSH Multi-Port Setup** (یا گزینه‌ای که اسکریپت multiport را اجرا می‌کند) را باز کنید.

۴. گزینه‌ها به صورت زیر است:

   * گزینه **Add port** — افزودن یک پورت (sshd + socket + UFW)
   
   * گزینه  **Remove port** — حذف یک پورت از پیکربندی و UFW
   
   * گزینه **Setup multiple ports (quick)** — وارد کردن فهرست جداشده با ویرگول (مثلاً `22, 3398, 443, 80, 8080`)؛ اسکریپت فاصله‌ها را حذف کرده و فقط پورت‌هایی را اضافه می‌کند که از قبل در پیکربندی نیستند.
   
   * گزینه **Verify setup** — بررسی پورت‌های در حال گوش‌دادن و پیکربندی

اسکریپت انتظار دارد **UFW** نصب باشد؛ اگر UFW موجود نباشد و نتواند آن را نصب کند، ادامه نمی‌دهد.

برای رفع مشکلات دیگر (مثل «Configuration validation failed»، قفل‌شدن دسترسی)، به [TROUBLESHOOTING.fa.md](TROUBLESHOOTING.fa.md) مراجعه کنید.
